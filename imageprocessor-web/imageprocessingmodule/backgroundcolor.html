BackgroundColor

Changes the background color of the current image.

Requests are passed thus: 

<!--Hex-->
http://your-image?bgcolor=800080
<!--RGBA-->
http://your-image?bgcolor=128,0,128,255
<!--Known Color-->
http://your-image?bgcolor=purple
