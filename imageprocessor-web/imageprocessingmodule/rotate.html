Rotate

Rotates the current image by the given angle without clipping.

Requests are passed thus:

{% highlight xml %} http://your-image?rotate=26 http://your-image?rotate=54&bgcolor=fff http://your-image?rotate=angle-54,bgcolor-fff {% endhighlight %}
