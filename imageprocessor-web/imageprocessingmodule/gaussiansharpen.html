GaussianSharpen

Uses a Gaussian kernel to sharpen the current image. Pass the desired kenel size to the processor.

Additional optional properties are also available
  • sigma (defaults to 1.4)
  • threshold (defaults to 0)

All gaussian sharpen requests have limiters to the given values set in the processing.config file to help prevent DoS attacks.

Requests are passed thus:

{% highlight xml %} http://your-image?sharpen=10&sigma=1.5&threshold=10 http://your-image?sharpen=10,sigma-1.5,threshold-10 {% endhighlight %}
