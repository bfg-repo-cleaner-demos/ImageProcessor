Vignette

Adds a vignette image effect to the current image.

Requests are passed thus:

{% highlight xml %} http://your-image?vignette=true http://your-image?vignette=800080 http://your-image?vignette=128,0,128,255 http://your-image?vignette=purple {% endhighlight %}
