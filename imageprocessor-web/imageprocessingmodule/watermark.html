Watermark

Adds a text based watermark to the current image with a wide range of options.

  • text
  • size (defaults to 48px)
  • font (any installed font)
  • opacity (1-100)
  • style (bold|italic|regular|strikeout|underline) defaults to regular
  • shadow
  • position (x,y)

Requests are passed thus: Each parameter is optional.

http://your-image?watermark=text-watermark goodness,color-fff,size-36,style-italic,opacity-80,position-30,150,shadow-true,font-arial
