Preset

ImageProcessor.Web allows you to configure preset commands to save on typing and centralize control. These commands are set in the processing.config file under the path \config\imageprocessor\ within your solution. Presets can be overwritten by appending additional commands. 

<?xml version="1.0" encoding="utf-8" ?>
<processing>
<plugins>
    <!-- Presets that allow you to reduce code in your markup. 
        Note the use of &#038; to escape ampersands. -->
    <plugin name="Preset">
    <settings>
        <setting key="presetname" value="width=300&#038;height=150"/>
    </settings>
    </plugin>
</plugins>
</processing>

Requests are passed thus: 

<--Standard-->
http://your-image?preset=presetname
<--Suffixed-->
http://your-image?preset=presetname&width=500
