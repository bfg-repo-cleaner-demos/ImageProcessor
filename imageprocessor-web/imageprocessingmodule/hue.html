Hue

Alters the hue of the current image changing the overall color. The angle can be and value between 0 and 360 degrees.

Added version 2.1.0

Requests are passed thus: 

<!--Hue-->
http://your-image?hue=180,height
<!--Hue Rotate-->
http://your-image?hue=180&rotatehue=true
