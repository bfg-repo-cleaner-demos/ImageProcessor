Format

Sets the output format of the current image to the matching ImageProcessor.Imaging.Formats.ISupportedImageFormat.

{% highlight c# %} public ImageFactory Format(ISupportedImageFormat format) {% endhighlight %}

Parameters

format
The ImageProcessor.Imaging.Formats.ISupportedImageFormat to set the image to.

Core formats are:

  • BitmapFormat
  • JpegFormat
  • GifFormat
  • PngFormat
  • TiffFormat
