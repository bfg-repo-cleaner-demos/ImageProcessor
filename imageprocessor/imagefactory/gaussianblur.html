GaussianBlur

Uses a Gaussian kernel to blur the current image.

{% highlight c# %} public ImageFactory GaussianBlur(int size) {% endhighlight %}

Parameters

size
The size of the kernel by which to blur the images pixels.
{% highlight c# %} public ImageFactory GaussianBlur(GaussianLayer gaussianLayer) {% endhighlight %}

Parameters

GaussianLayer
The ImageProcessor.Imaging.GaussianLayer containing the following properties required to blur the image.
Size
The size to set the Gaussian kernel to.
Sigma
The Sigma value (standard deviation) for Gaussian function used to calculate the kernel.
Threshold
The threshold value, which is added to each weighted sum of pixels.
