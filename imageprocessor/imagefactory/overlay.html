Overlay

Adds a image overlay to the current image. If the overlay is larger than the image it will be resized to match the images dimensions.

Added version 2.1.0

public ImageFactory Overlay(ImageLayer imageLayer)

Parameters

imageLayer
The ImageProcessor.Imaging.ImageLayer containing the properties necessary to add the image overlay to the image.
Image
The System.Drawing.Image overlay the image with.
Size
The System.Drawing.Size at which to render the overlay.
Opacity
The opacity of the image. Between 0 and 100
Position
The Nullable<System.Drawing.Point> determining the position within the current image to render the overlay. If not set then the overlay will be centered.
