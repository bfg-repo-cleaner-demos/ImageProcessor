Hue

Alters the hue of the current image changing the overall color.

Added version 2.1.0

public ImageFactory Hue(int degrees, bool rotate = false)

Parameters

degrees
The angle by which to alter the images hue. Any integer between 0 and 360.
rotate
Whether to rotate the hue of the current image altering each color. Defaults to False
