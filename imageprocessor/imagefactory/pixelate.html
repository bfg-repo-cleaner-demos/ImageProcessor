Pixelate

Pixelates an image with the given size.

Added version 2.1.0

public ImageFactory Pixelate(int pixelSize, Rectangle? rectangle = null)

Parameters

pixelSize
The size of the pixels to create.
rectangle
The Nullable<System.Drawing.Rectangle> representing the area in which to pixelate the image. If not set, the whole image is pixelated.
